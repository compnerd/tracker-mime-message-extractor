tracker-mime-message-extractor by compnerd

A MIME Message Extractor for Tracker

Dependencies

gmime:2.6[>=2.5.1] tracker:0.8

Install

cd /path/to/sources ./autogen.bash ./configure --prefix=/usr --sysconfdir=/etc --localstatedir=/var/lib make make install
